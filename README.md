# Audio Switcher
A lightweight Windows 11 system tray app that toggles your default audio output between two devices with a global hotkey. Zero dependencies, just a single .exe.

## Download
Grab AudioSwitcher.exe from the Releases page. No installation or .NET runtime needed.

## Usage
1. Run AudioSwitcher.exe

2. Right-click the tray icon → Select Devices to pick your two audio outputs

3. Press Ctrl + Alt + S to toggle between them

## Features
- Global hotkey toggle (customizable via tray menu)

- System tray icon with right-click menu

- Toast notification showing the active device

- Settings saved between restarts

- Optional start with Windows

- About dialog with version info and credits

- No NuGet packages or external dependencies
